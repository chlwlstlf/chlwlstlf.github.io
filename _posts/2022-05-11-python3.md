# 파이썬 객체지향 방식

## Problem 1.

```python
class scores :
  def __init__(self, name, k, e, m):
    self.name = name
    self.k = k
    self.e = e
    self.m = m

  def avg(self):
    return (self.k + self.e + self.m)/3

s1 = scores("홍길동", 50, 70, 90)

print("이름: %s" %s1.name)
print("국어: %d, 영어: %d, 수학: %d" %(s1.k, s1.e, s1.m))
print("평균: %.1f" %s1.avg())
```

    이름: 홍길동
    국어: 50, 영어: 70, 수학: 90
    평균: 70.0

## Problem 2.

```python
class Ladder:
  def __init__(self, under, upper, height):
    self.under = under
    self.upper = upper
    self.height = height

  def area(self):
    return (self.under + self.upper)/2 * self.height


w1 = int(input("사다리꼴 밑변의 길이 입력: "))
w2 = int(input("사다리꼴 웻변의 길이 입력: "))
h = int(input("높이 입력: "))
ladder1 = Ladder(w1, w2, h)
print("사다리꼴의 면적: ", ladder1.area())
```

    사다리꼴 밑변의 길이 입력: 10
    사다리꼴 웻변의 길이 입력: 20
    높이 입력: 3
    사다리꼴의 면적:  45.0
